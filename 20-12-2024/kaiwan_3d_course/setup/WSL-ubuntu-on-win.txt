*WSL*
Windows Subsystem for Linux

With Windows 10 and 11, it's entirely possible to install Ubuntu directly on Win - via WSL!
 - Windows Susbsystem for Linux
 Win 11 (or later) will be easier (than on Win 10)..

 Ref:
 - Install Ubuntu on Windows 11 fast using WSL2: https://www.youtube.com/watch?v=1kFFwknneD8 
 - Run Ubuntu 24.04 on Windows 11 with GUI using WSL: https://kskroyal.com/run-ubuntu-24-04-on-windows-11-with-gui-using-wsl/
 
 
 Once Ubuntu's installed via WSL, do this:
 
 sudo apt update
 sudo apt upgrade
 sudo apt install build-essential
